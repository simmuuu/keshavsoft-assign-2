# Setting up Project
1. Clone the repo
```
git clone https://github.com/simmuuu/keshavsoft-assign-2.git
cd keshavsoft-assign-2
```

2. Install dependencies
```
pnpm install
```

3. Build
```
pnpm build
```

4. Development server:
```
pnpm dev
```
